# BlueStar AI エージェントカタログ

## アーキテクチャ原則: プログレッシブディスクロージャー
ユーザーは最初から詳細なエージェントを指定する必要はありません。まずは「司令塔（Orchestrator）」となるエージェント（PM, TechLead等）に指示を出し、そこから必要に応じて専門エージェント（Specialist）や作業エージェント（Utility）が呼び出される構成を目指します。

---

## 1. プロダクト・企画 (Plan & Define)

| エージェント名 | 役割属性 | 概要・役割 | 優先度 |
| :--- | :--- | :--- | :--- |
| **プロダクトマネージャー (PdM)** | Orchestrator | **【司令塔】** 要件定義、ロードマップ策定、各専門エージェントへの指示出しを行う。ユーザーとの対話窓口。 | 高 |
| **UXリサーチャー** | Specialist | ターゲットユーザーになりきって、機能や記事に対するフィードバックやニーズ出しを行う。 | 中 |
| **ビジネスアナリスト** | Specialist | 競合調査、市場分析を行い、プロダクトの差別化要素を言語化する。 | 低 |

## 2. 開発・エンジニアリング (Build)

| エージェント名 | 役割属性 | 概要・役割 | 優先度 |
| :--- | :--- | :--- | :--- |
| **テックリード** | Orchestrator | **【司令塔】** 技術選定、アーキテクチャ設計、コード品質基準の策定。コーダーへの指示書（Spec）を書く。 | 高 |
| **フロントエンド・コーダー** | Specialist | UI/UX、Component設計、State管理に特化して実装する。Tailwind/React等のスタックに特化。 | 高 |
| **バックエンド・コーダー** | Specialist | API設計、DB設計、ビジネスロジック実装に特化する。セキュリティ・パフォーマンス重視。 | 高 |
| **QA/テストエンジニア** | Specialist | テストケースの作成、自動テストコード（E2E/Unit）の実装を行う。意地悪な視点を持つ。 | 中 |
| **コードレビュー** | Utility | セキュリティ脆弱性、可読性、命名規則の観点でコードをレビューする。 | 高 |

## 3. 運用・インフラ (Run & Ops)

| エージェント名 | 役割属性 | 概要・役割 | 優先度 |
| :--- | :--- | :--- | :--- |
| **SRE / インフラアーキテクト** | Orchestrator | **【司令塔】** クラウド構成（AWS/GCP）、IaC（Terraform）の設計と構築。 | 中 |
| **DevOps** | Utility | GitHub Actionsなどのパイプライン構築、デプロイフローの整備。 | 中 |
| **セキュリティ監査** | Utility | インフラ設定やコードの依存関係における脆弱性（CVE）を定期チェックする。 | 低 |

## 4. 広報・ブランディング (Promote)

| エージェント名 | 役割属性 | 概要・役割 | 優先度 |
| :--- | :--- | :--- | :--- |
| **編集長 (Editor in Chief)** | Orchestrator | **【司令塔】** ブランディング戦略に基づき、記事のネタ出し、スケジュール管理、公開判定を行う。 | 中 |
| **テックブログ記述** | Specialist | 技術的な内容を、ターゲット読者に合わせて分かりやすく執筆する。 | 中 |
| **テックブログレビュー** | Utility | ファクトチェック、図解推奨、SEO観点でのチェックを行う。 | 中 |
| **SNSマーケター** | Specialist | ブログ記事を元に、X（Twitter）やLinkedIn用の要約・告知ポストを作成する。 | 低 |

## 5. 基盤・ナレッジ (Meta & Knowledge)

| エージェント名 | 役割属性 | 概要・役割 | 優先度 |
| :--- | :--- | :--- | :--- |
| **プロンプトエンジニア** | Specialist | 各エージェントのシステムプロンプトのバージョン管理、改善提案を行う。 | 高 |
| **ドキュメンタリアン** | Utility | 会話やコードの変更履歴から、社内Wiki（Notion等）を自動更新する。 | 低 |
| **コンシェルジュ** | Interface | 社員がどのエージェントを使えばいいか迷った時の案内係。 | 中 |
