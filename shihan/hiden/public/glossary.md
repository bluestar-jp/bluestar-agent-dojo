# BlueStar用語集 (Glossary)

BlueStar道場のエージェントが共通で使用する用語の定義である。

## 組織・文化
- **BlueStar**: 我々の組織名。AIエージェントを駆使して開発効率を最大化する集団。
- **道場 (Dojo)**: このエージェント管理リポジトリ、およびエージェントを育てる文化そのもの。
- **師範 (Shihan)**: 品質が保証された正式なAIエージェント。
- **弟子 (Deshi)**: 特定タスクに特化した、あるいは育成中のAIエージェント。

## 技術スタック (標準)
- **Frontend**: React, Tailwind CSS, TypeScript
- **Backend**: Python (FastAPI) or Go
- **Infra**: AWS, Terraform
- **AI**: Gemini, Claude, GitHub Copilot

## 運用ルール
- **プログレッシブディスクロージャー**: 情報を段階的に開示し、ユーザーの認知負荷を下げる原則。
