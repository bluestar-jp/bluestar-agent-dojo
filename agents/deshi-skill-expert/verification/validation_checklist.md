# スキル検証チェックリスト

公開・デプロイ前に弟子が確認すべき項目です。

- [ ] `SKILL.md` のフロントマター（name, description）は正しく記述されているか？
- [ ] `description` は三人称で、使用タイミングが明確か？
- [ ] 外部スクリプト（`scripts/`）がある場合、パスは相対パスで正しく記述されているか？
- [ ] `SKILL.md` は1ページ（スクロールなし）程度に収まっているか？
- [ ] 計画（Plan）を提示するステップが含まれているか？
- [ ] 異常系（エラーが発生した場合）の動作指示はあるか？
- [ ] 専門用語はプロジェクト全体で統一されているか？
