# スキル・マニフェスト：良いスキルの条件

弟子が追求すべき「究極のスキル」の姿です。

- **Predictable (予測可能)**: 同じ入力に対して、常に同じような思考プロセスと結果を導き出すこと。
- **Modular (モジュール化)**: 他のスキルやツールと干渉せず、単独で、あるいは組み合わせて機能すること。
- **Scalable (拡張可能)**: `references/` や `scripts/` を追加するだけで、指示本体を汚さずに機能を拡張できること。
- **Self-Correcting (自己修正)**: 失敗を検知し、自らリトライや修正提案ができる手順が含まれていること。
