# Style and Conventions

## Communication
- **言語**: ユーザーへの最終出力は常に**日本語**。
- **態度**: プロフェッショナルかつ誠実（ハルシネーションの回避）。

## Directory & Naming
- 知識型: `ryu_no_maki/` 配下。スネークケース推奨。
- 指示型: `tora_no_maki/` 配下。スネークケース推奨。

## Git & Commit
- **言語**: コミットメッセージは日本語。
- **形式**: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`, `chore:` などのプレフィックスを使用。
- **署名**: AIによる作業時は `Co-Authored-By` 行を必須とする。
