# BlueStar Agent Dojo - Global Instructions

本ファイルは、このリポジトリ内で活動する全てのAIエージェントに対する最上位の指示書である。

## 1. 憲法とルールの遵守
あなたは作業を開始する前に、必ず以下のファイルを確認し、その指示に従わなければならない。
- `shihan/okite/Constitution.md`: BlueStarの最上位憲法。
- `AGENTS.md`: エージェントの役割定義と連携プロトコル。

## 2. コミットルールの徹底 (重要)
コードやドキュメントの変更をコミットする際は、以下のルールを厳守せよ。
- **言語**: 日本語
- **フォーマット**: `<prefix>: <内容>` (Conventional Commits準拠)
- **署名**: 末尾に必ず以下を付与する。
  ```text
  Co-Authored-By: gemini-cli <218195315+gemini-cli@users.noreply.github.com>
  ```

## 3. 秘伝（コンテキスト）の活用
回答の質を上げるため、`shihan/hiden/` 配下にあるドキュメント（用語集等）を積極的に参照し、BlueStar固有の文脈を理解せよ。

## 4. プログレッシブディスクロージャー
ユーザーへの回答は常に段階的に行え。
1. 概要と方針の提示（合意形成）
2. 詳細な設計や実装（具体的な作業）
3. 動作確認と最終報告
一度に大量の情報を提示してはならない。
