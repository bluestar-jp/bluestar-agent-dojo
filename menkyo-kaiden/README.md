# 免許皆伝 (Menkyo-Kaiden)

**【鍛錬中...】**

このディレクトリは、**「師範 (Shihan)」**として定義されたエージェントの人格やルールを、各AIツールが実際に利用可能な**拡張機能（Extension / Plugin / Config）**として提供するためのパッケージ置き場です。

## コンセプト
道場の「正本（Single Source of Truth）」である `shihan/` ディレクトリの内容を、以下の各ツールの作法に合わせて変換・展開します。

- **Gemini CLI**: `.gemini/extensions/` に配置可能な Extension パッケージ。
- **Claude Code**: プロジェクトコンテキストとして読み込めるプロンプト定義。
- **Cursor / VS Code**: エディタ用のルールファイル (`.cursorrules` 等)。

## 現状とロードマップ

将来的には、`shihan/` のMarkdown定義から、各ツール用の設定ファイルを自動生成・同期する仕組み（ビルドパイプライン）を構築する予定です。
